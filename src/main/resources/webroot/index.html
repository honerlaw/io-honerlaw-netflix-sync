Netflix Sync
Chrome Store Extension Repository Server Repository

Netflix Sync is a Google Chrome Extension that allows users to easily sync videos on Netflix and watch them together.

The goal behind Netflix Sync is to simply allow multiple users to watch videos together without being in the same room. This extension was created in order to allow people in long distance relationships, or friends who may not be able to get together because of distance, to watch movies together.
